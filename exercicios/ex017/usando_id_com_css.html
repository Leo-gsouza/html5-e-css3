ID

Usando o ID

O id='principal' permite diferenciar a configuração de uma TAG semelhante, exemplo:
h1 id="principal" vai ter uma configuração diferente do h1>

ID e #

No HTML será usado o id na frente da TAG, ja no CSS é usado o #antes do nome dado para o id, exemplo:

HTML = «h1 id="principal"»CRIANDO SITE COM HTML E CSS

CSS = h1#principal{color: #3C3FBB; text-align: center;

ID é unico

Não pode usar o mesmo id para duas TAGS diferentes
