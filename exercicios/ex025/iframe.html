Dicas iframe

  1. Iframe não é o frame usado no HTML anterior
  2. Iframe não é horrivel e nem perfeito
  3. O google boot talvez não consiga entrar nos iframes
  4. Problemas com acessibilidade - pessoas cegas
  5. Problemas com usabilidade - Pode deixar as pessoas confusas

sandbox e referrerpolicy

Usando sandobox="sandbox" você protege que outros sites consigam coletar informações fornecidas

Usando o referrerpolicy="no-referrer" O que está dentro do iframe não vai coletar nenhum dado do cliente

recomendado usar os dois sandbox e referrerpolicy

caso deseje que o usuario digite dados no iframe tem que usar uma segurança com nivel inferior do sandbox que é allow-same-origin allow-forms allow-script, com isso é permitido mesma origem, formularios e scripts
